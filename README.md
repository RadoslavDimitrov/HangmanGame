# HangmanGame
Simple Hangman console game. Here i just practice my C# skills and my skills with Json files.

## Technologies
* .Net 5
* Local Json file as DB

## How to play
* Clone my repo.
* Start Hangman.sln.
* DB will be empty, so you will need to add few words first.
* Enjoy.
